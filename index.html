자바스크립트의 특징

자바스크립트는 HTML, CSS와 함께 쓰여 웹페이지를 완성하고, 버튼 클릭이나 화면 동작 같은 기능을 구현할 수 있는 언어이다.

계산기 만들기 체험

  • HTML로 버튼과 입력창 만들기
  • CSS로 버튼과 화면 꾸미기
  • JavaScript로 사칙연산 기능 구현
  • 실제 동작하는 계산기를 브라우저에서 테스트

프로그래밍 언어 비교

  • Python: 쉬운 문법, 웹 개발 및 데이터 분석에 활용
  • C: 빠른 속도, 시스템 프로그래밍과 임베디드에 사용
  • Java: 객체 지향, 모바일 앱과 대규모 프로젝트에 활용
  • JavaScript: 웹페이지 기능 구현과 인터랙티브 요소 담당
HTML 역할 보기

HTML은 웹페이지의 뼈대와 구조를 만드는 언어로, 글, 이미지, 버튼 등 모든 요소의 위치와 내용을 정의합니다.

CSS 역할 보기

CSS는 웹페이지를 꾸며주는 언어로, 색상, 글꼴, 레이아웃, 애니메이션 등 디자인과 스타일을 담당합니다.
